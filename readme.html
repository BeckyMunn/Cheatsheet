Style Syntax Example

Image Links
Context Relative Link

Lists
Style Syntax Example
Unordered List Dash - - George Washington
Unordered List Dot * * George Washington
Ordered List 1. 1. George Washington
Nested List Until the list marker character lies directly below the first character of the text in the item above it. 
1. First list item
   - First nested list item
     - Second nested list item
Task List - [ ] - [ ] Do a task
